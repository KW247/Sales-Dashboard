# Sales Data Reporting Dashboard

## Overview
This project focuses on processing and reporting transactional sales data to support business monitoring and decision-making.

## Tools Used
- SQL (data extraction and aggregation)
- Power BI (dashboard development)
- Excel (data cleaning)

## Key Tasks
- Extracted and aggregated sales data using SQL
- Cleaned and structured transactional datasets
- Performed validation checks to ensure data consistency
- Built an interactive Power BI dashboard to track key metrics

## Key Metrics
- Revenue
- Order volume
- Product performance
